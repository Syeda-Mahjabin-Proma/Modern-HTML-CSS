## Overview

This project demonstrates a **responsive layout using CSS Flexbox**. It features a **header, horizontal navigation menu, main content section, and a sidebar**. 
The layout highlights **flexbox properties** to create flexible and structured page sections, while showcasing basic **typography, spacing, and hover effects**.

A live demo is available here: [Flexbox Layout Challenge](https://syeda-mahjabin-proma.github.io/Modern-HTML-CSS/4_flex_box/)
