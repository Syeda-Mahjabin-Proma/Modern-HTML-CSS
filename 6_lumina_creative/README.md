## Overview

This project showcases a **multi-page creative agency website** built using **HTML5, CSS3, and Lightbox for image galleries**. It includes
**Home, About, and Contact pages**, featuring a **hero section, responsive gallery, services overview, team section, and contact form**. 
The layout uses **flexbox and responsive design principles** to ensure it adapts seamlessly across different screen sizes, with 
consistent **typography, colors, and hover effects** for a modern, professional look.

Live demo: [Lumina Creative Website](https://syeda-mahjabin-proma.github.io/Modern-HTML-CSS/6_lumina_creative/)
