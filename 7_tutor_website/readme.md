# **Overview**

This project showcases a **responsive online course landing page** built using **HTML5, CSS3, and 
vanilla JavaScript**. It includes **Home and Contact pages**, featuring a
**hero section, course chapters, key takeaways, author info, newsletter signup, and 
contact form with map**. The layout uses **flexbox, grid, and responsive design principles** to 
ensure it adapts seamlessly across different screen sizes, with consistent
**typography, colors, and hover effects** for a modern, professional look.  

Live demo: [Tutor Landing Page](https://github.com/Syeda-Mahjabin-Proma/Modern-HTML-CSS/7_tutor_website)
