## Overview:

Leno App is a mobile-first productivity and health companion designed for developers and creators. It helps users track goals, manage tasks, and stay focused
through features like real-time data, calendar integration, and clean reading interfaces. The app includes interactive previews, detailed analytics, and multiple
pricing options. Built with HTML, CSS, JavaScript, and responsive design, it ensures a smooth experience across devices. The project also implements a mobile
menu and video modal for enhanced interactivity.
