# Modern HTML & CSS Practice Projects 🚀

This repository contains my hands-on practice projects created while learning modern web development using **HTML5 and CSS3**.
Each folder represents a small project or concept designed to strengthen my understanding of layout systems, forms, responsive design, and core styling techniques.

These projects were built as part of my learning journey in modern front-end development.

---

## 📁 Project Overview

This repository includes multiple beginner-to-intermediate level projects focusing on:

* Semantic HTML5 structure
* CSS3 styling and layout techniques
* Flexbox fundamentals
* Responsive design concepts
* Form design and input handling
* Reusable components and clean structure
* Practical mini-project implementation

## 🧑‍💻 How to Run the Projects

1. Clone the repository:

```bash
git clone https://github.com/Syeda-Mahjabin-Proma/Modern-HTML-CSS.git
```

2. Navigate to the project folder:

```bash
cd Modern-HTML-CSS
```

3. Open any project folder.

4. Open the `index.html` file in your browser.
