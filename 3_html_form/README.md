## Overview

This project demonstrates a **basic HTML form** using standard form elements such as **text, number, range, checkbox, date, and submit inputs**. 
It showcases **semantic structure** with `<header>` and `<main>` tags, providing a simple example of **how to create and organize an HTML form**.

Live demo: [View Form](https://syeda-mahjabin-proma.github.io/Modern-HTML-CSS/3_html_form/)
